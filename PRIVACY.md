# Privacy

Telegram Pixel Dungeon does not include analytics, advertising, payments or a custom
tracking backend.

Game progress and settings are saved locally in the browser. When the game is
opened as a Telegram Mini App and Telegram CloudStorage is available, the same
save data is also synchronized to the current Telegram user's CloudStorage so
that progress can be restored on another device. The Telegram user ID is used
only to separate local save namespaces on the device; this project does not
send it to a developer-operated server.

Deleting the Mini App's site data removes the local copy. Telegram-managed
cloud data is subject to Telegram's platform controls and privacy policy.
