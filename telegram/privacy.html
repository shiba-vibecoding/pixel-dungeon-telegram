Pixel Dungeon — Privacy

Last updated: 22 July 2026

This Mini App has no analytics, advertising, payments or developer-operated tracking server.

Progress and settings are stored in the browser. Inside Telegram, supported clients also synchronize the same data with the current user's Telegram CloudStorage so it can be restored on another device. The Telegram user ID is used only to keep local saves separated on the device and is not sent to a server operated by this project.

Clearing the Mini App's site data removes the local copy. Cloud data is stored and governed by Telegram. For privacy questions or a cloud-data deletion request, contact @barboskich.

Конфиденциальность

Обновлено: 22 июля 2026 года

В мини-приложении нет аналитики, рекламы, платежей и собственного сервера слежения.

Прогресс и настройки хранятся в браузере. В Telegram на поддерживаемых клиентах те же данные дополнительно синхронизируются с Telegram CloudStorage текущего пользователя, чтобы восстановить игру на другом устройстве. Telegram ID используется только для разделения локальных сохранений на устройстве и не отправляется на сервер этого проекта.

Очистка данных сайта мини-приложения удаляет локальную копию. Облачные данные хранит Telegram по своим правилам. По вопросам конфиденциальности или для запроса удаления облачных данных напишите @barboskich.
